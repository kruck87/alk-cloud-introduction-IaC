Projekt przygotowany jako IaC. Projekt wykorzystuje ponisze elementy

  • Utworzenie sieci VPC, wraz z subnetem
  • Stworzenie template do Instances group
  • W ramach template'u skrypt startowy instalujący apache, tworzący pomocniczne pliki html
  • Główny plik - index.html pobierany jest z bucketu
  • Utworzono VM jako MIG - na podstawie template. Zastosowano autoscaling i health check
  • Skonfigurowano publiczne uprawnienia bucketów i obiektów
  • Utworzenie Loadbalancera (1 front, Backendy: MIG oraz bucket) - mozna sprawdzić wchodząc kilkukrotnie w link
  • Utworzenie URL Map
  • Utworzenie bucketu przechowującego media statyczne (obrazki itp)
  • Zarezerwowanie statycznego IP dla Load Balancera
  • Bucket z mediami cachowany jest za pomocą Cloud CDN - wszystkie pliki dostępne po ściezce /src są cacheowane
  • Skonfigrowano NAT
